# рҹ’¬ Chatify вҖ” AI-Powered Real-Time Chat Application

<div align="center">

A full-stack real-time chat application built with the **MERN stack** (MongoDB, Express, React, Node.js) featuring **AI-powered chat intelligence** using Groq's Llama 3.1 model.

**Real-time messaging** В· **AI Summarization** В· **Smart Replies** В· **Sentiment Analysis** В· **Voice-to-Text**

## вңЁ Features

### рҹ’¬ Core Chat
| Feature | Description |
|---------|-------------|
| **Real-time messaging** | Instant message delivery via Socket.IO |
| **Image sharing** | Upload & send images via Cloudinary CDN |
| **Online status** | See who's online in real-time |
| **Typing indicators** | Know when someone is typing |
| **Notification sounds** | Audio alerts for new messages |

### рҹӨ– AI-Powered Features (Groq + Llama 3.1)
| Feature | Description |
|---------|-------------|
| **вңЁ Chat Summarization** | Summarize entire conversations in 10+ languages |
| **вҡЎ Smart Replies** | Gmail-style AI reply suggestions after every message |
| **рҹҳҠ Sentiment Analysis** | Real-time mood detection with emoji indicators |
| **рҹ”Қ AI Chat Search** | Ask natural language questions about chat history |
| **рҹ“Ӣ Meeting Notes** | Auto-generate structured meeting minutes with agenda, decisions & action items |
| **рҹҺҷпёҸ Voice-to-Text** | Browser speech recognition вҶ’ text input |
| **рҹ”‘ Keyword Highlights** | Auto-highlight dates, deadlines, and tasks in messages |

### рҹ”җ Security & Auth
| Feature | Description |
|---------|-------------|

| **Password hashing** | bcrypt with salt rounds |
| **Rate limiting** | Arcjet-powered API protection |
| **Socket auth** | WebSocket connections require JWT |

## рҹҡҖ Getting Started

### Prerequisites
- **Node.js** v20.0.0 or higher
- **MongoDB** (Atlas or local)
- **Cloudinary** account (image uploads)
- **Groq** API key (free at [console.groq.com](https://console.groq.com))

### 1. Clone the repository
```bash
git clone https://github.com/yourusername/chatify.git
cd chatify-master
```

### 2. Setup backend environment
```bash
cp backend/.env.example backend/.env
```

Edit `backend/.env` and fill in your values:
```env
PORT=3000
MONGO_URI=mongodb+srv://user:pass@cluster.mongodb.net/chatify
NODE_ENV=development
JWT_SECRET=your_super_secret_key
RESEND_API_KEY=re_xxxxxxxxxxxx
EMAIL_FROM=noreply@yourdomain.com
EMAIL_FROM_NAME=Chatify
CLIENT_URL=http://localhost:5173
CLOUDINARY_CLOUD_NAME=your_cloud
CLOUDINARY_API_KEY=123456789
CLOUDINARY_API_SECRET=your_secret
ARCJET_KEY=ajkey_xxxxxxxxxxxx
ARCJET_ENV=development
GROQ_API_KEY=gsk_xxxxxxxxxxxx
```

### 3. Install dependencies
```bash
# Install backend dependencies
cd backend
npm install

# Install frontend dependencies
cd ../frontend
npm install
```

### 4. Run the application

**Terminal 1 вҖ” Backend:**
```bash
cd backend
npm run dev
```
> Starts on `http://localhost:3000`

**Terminal 2 вҖ” Frontend:**
```bash
cd frontend
npm run dev
```
> Starts on `http://localhost:5173`

### 5. Production build
```bash
# From root directory
npm run build   # Builds both backend & frontend
npm start       # Serves production build
```
